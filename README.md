# OmniNexus: Enterprise Intelligence & Automation Suite

OmniNexus is a comprehensive, multi-modal enterprise platform designed to bridge the gap between traditional financial operations and autonomous artificial intelligence.

## Structure

- `apps/web`: React-based Frontend (Vite + TailwindCSS)
- `apps/api`: FastAPI Backend
- `apps/worker`: Background workers (Celery/Arq)
- `apps/agents`: LangGraph Agent Swarm
- `packages/`: Shared libraries
